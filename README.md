# SkylineSystem

## Features(x86_64)

* Devices
  * AHCI
  * IDE
  * ACPI
  * IOAPIC
  * LAPIC
* Basics
  * Syscall
  * Scheduler
  * Output
  * SMP
  * Virtual Memory
* Disk Interfaces
  * Sata
  * Ram

Many more in future
